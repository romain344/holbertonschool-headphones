# Headphones — Holberton School Web Project

## About the Project

This project is a fully responsive web page built from scratch using pure HTML and CSS — no external libraries or frameworks. The design is based on a Figma file provided by Holberton School and faithfully reproduces a modern, clean landing page for a fictitious headphones brand. Every section, layout, and visual detail was implemented by hand, from the hero banner down to the footer, following the design specifications pixel by pixel.

The page adapts seamlessly to any screen size. On screens wider than 480px, users get the full desktop experience with a multi-column layout and a fixed navigation bar. At 480px and below, the layout switches to a single-column mobile view, ensuring readability and usability on all devices. Interactive elements such as links and buttons respond to hover and active states with color and opacity changes, giving the page a polished, professional feel.

The content area is capped at 1000px and centered horizontally, keeping the design visually balanced on large monitors without stretching across the full viewport width.
